#AI-Driven Healthcare Anomaly Detection System

## Project Overview
This is an end-to-end real-time monitoring platform designed to detect abnormal patterns in patient vital signs. It helps identify health risks early by analyzing data like Heart Rate and SpO2.

## Key Features
- **Real-time Detection:** Uses Isolation Forest for anomaly detection.
- **Severity Classification:** Categorizes risks into LOW, MEDIUM, and HIGH.
- **Interactive Dashboard:** Built with Streamlit for easy visualization.
- **Alert System:** Visual triggers for critical health conditions.

## Tech Stack
- **Language:** Python
- **AI/ML:** Scikit-learn (Isolation Forest)
- **Frontend:** Streamlit
- **Data:** Pandas, Numpy

## How to Run
1. Install dependencies: `pip install -r requirements.txt`
2. Run the app: `streamlit run app.py`
